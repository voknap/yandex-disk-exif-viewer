# Yandex Disk EXIF Viewer 📸📍

Удобное расширение для браузера Chrome, которое позволяет мгновенно просматривать EXIF-данные фотографий (модель камеры, параметры съемки, дату) прямо в интерфейсе Яндекс Диска, а также показывает место съемки на интерактивной карте.

### Основные возможности:
*   **Полный EXIF**: просмотр модели камеры, объектива, ISO, выдержки, диафрагмы и многого другого.
*   **Интерактивная карта**: если в фото есть GPS-координаты, расширение покажет место съемки на карте (доступно 7 различных слоев карт).

### Инструкция по установке в Google Chrome:

1.  **Скачайте проект**: Сохраните файлы расширения в отдельную папку на вашем компьютере.
2.  **Откройте расширения**: Введите в адресной строке Chrome `chrome://extensions/` и нажмите Enter.
3.  **Режим разработчика**: Включите переключатель **"Режим разработчика"** (Developer mode) в правом верхнем углу.
4.  **Загрузите расширение**: Нажмите кнопку **"Загрузить распакованное расширение"** (Load unpacked) слева сверху.
5.  **Выберите папку**: Укажите путь к папке, в которой лежат файлы проекта (там, где находится файл `manifest.json`).

**Готово!** Теперь при просмотре фотографий на Яндекс Диске у вас появится кнопка для просмотра EXIF и карты. 🚀🛰️🗺️
